## 🔍 Finding Your Class Files

### Regular Session Classes

Files are organised under the relevant year and term folder. Each class folder follows this naming format:

```
L[level][day code][term] - [day]
```

**Level** — The `L` followed by digits indicates your level (e.g. `L03` = Level 3, `L4` = Level 4).  
**Day code** — A number indicating which day group you belong to (e.g. `08` = Saturday, `09` = Tuesday, `10` = Sunday).  
**Term** — `T` followed by a number indicates the term (e.g. `T3` = Term 3).

**Example:** `L0309T3 - Tues` → Level 3, Tuesday group, Term 3 session files.

**Terms run as follows:**
| Term | Period |
|------|--------|
| T1 | Jan – Mar |
| T2 | Mar – May |
| T3 | Jul – Sep |
| T4 | Sep – Nov |

### Competition Class

Files for competition-focused sessions are in the `Competition Class` folder under the relevant year, and in the top-level `Competitions/` folder. This includes:
- **HCIC** practice materials
- **Bebras** side code

### Holiday Camps

Files for holiday programmes are in the `Holiday Camps/` folder:
- **Basic Python** — introductory Python holiday programme
- **Advanced Algorithms** — advanced holiday programme

### External Classes

Files for classes run outside of TTA (e.g. NTU, St. Andrew's) are in the `External Classes/` folder.

## 📝 Notes

- Make-up session files are stored in a `Make up` subfolder within each class folder.
- Files may be Jupyter Notebooks (`.ipynb`) or Python scripts (`.py`) depending on the class level and topic.
